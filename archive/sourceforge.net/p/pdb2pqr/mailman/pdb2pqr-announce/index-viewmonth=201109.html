[Pdb2pqr-announce] PDB2PQR 1.7.1a
From: Baker, Nathan <Nathan.B<aker@pn...> - 2011-09-13 16:33:49 
Dear PDB2PQR users --

We are pleased to announce the release of PDB2PQR 1.7.1a.  This release contains bug fixes which are described in detail at http://goo.gl/QOaaQ. Among these changes are several important fixes which should be of broad interest to the community. 

*	Added a force field example.
*	Fixed ligand command line option.
*	Fixed small bug in PQR output.
*	Fixed several errors in the web interface.
*	Updates to examples and documentation.

The new version of PDB2PQR can be downloaded from http://www.poissonboltzmann.org/pdb2pqr/d 

Thank you for your continuing support of PDB2PQR.

Sincerely,

The PDB2PQR Development Team
