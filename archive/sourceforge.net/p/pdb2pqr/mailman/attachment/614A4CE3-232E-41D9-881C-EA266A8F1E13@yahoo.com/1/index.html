As for whether it is “reuse”, these names have long been used in the molecular modeling community to represent residue states. The PDB knows this, but decided to use these names any way decades later. Modelers have used CYX since at least 1982 ( http://pubs.acs.org/doi/abs/10.1021/ja00387a046 ), but the PDB only started using it in 2003 and has only used it for a single structure.
