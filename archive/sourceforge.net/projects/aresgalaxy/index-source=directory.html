Editor's Review

Ares Galaxy is a peer-to-peer (P2P) BitTorrent file-sharing program that lets you quickly publish and share digital files including music, videos, photos, games, software, etc. with a single click. Members of the Ares social network can join or host chat rooms, creating a worldwide file-sharing hub. Because Ares Galaxy supports BitTorrent, you can preview the contents of files before you download them. You can also surf the Web directly within the program. And Ares’ completes jobs in a timely manner while keeping CPU and memory usage to a minimum. It also allows you to download files from multiple sources, which enhances transfer time.

Ares Galaxy at a Glance

Installing and launching Ares Galaxy took mere minutes, with its uncomplicated installation wizard and easy-to-navigate user interface (UI). Finding content is as simple as selecting the Search tab, typing the file name, and choosing a content type to help narrow search results and speed search time. Once you find what you seek, you can effortlessly download it with a double click. There are numerous chat channels to choose from in the IRC network. And the Control Panel tab lets you configure everything from the network proxy to the skin (appearance) you select.

Product Details

Here is a detailed breakdown of Ares Galaxy’s features:

  • Advanced search—Ares allows you to search for all types of media files such as audio, video, games, software, and photos.
  • Quick downloads—Ares automatically finds more sources and downloads files from multiple users. This can increase remarkably the speed of your downloads. Ares also works behind firewalls and requires no configuration.
  • Built in audio/video player—You can preview A/V files while downloads are in progress and organize a playlist of your favorites. And Ares' player supports Shoutcast Internet radio.
  • Chat rooms—You can host a chat room and/or join available channels while downloading files.
  • Powerful library organizer—Your shared files are organized into categories in a library section. You can sort your files by type, category and easily change share settings. And a handy quick-search feature locates your files by typing a keyword.

Pros

Cons
It is easy to quickly download and preview multiple media files and participate in chat rooms and file sharing. Using the Control Panel tab to configure the network settings might confuse the average user.

Description

Ares Galaxy is a free, open source BitTorrent and chat program that uses its own decentralized supernode/leaf network. Ares Galaxy has a simple, quick access interface with a built in audio/video viewer. Ares allows you to share any digital file including images, audio, video, software, documents, etc. You may now easily publish your files through the Ares’ peer-to-peer (P2P) network. As a member of the virtual community, you can search and download just about any file shared by other users. The latest versions also support the BitTorrent protocol and Shoutcast radio stations.
