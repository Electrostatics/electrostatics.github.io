Description

The Chemistry Development Kit (CDK) is a scientific, LGPL-ed library for bio- and cheminformatics and computational chemistry written in Java.

The Chemistry Development Kit Web Site

Categories

3D Rendering, Bio-Informatics, Chemistry, Medical Science Apps., Software Development

License

GNU Library or Lesser General Public License version 2.0 (LGPLv2)

Features

  • Read and write chemical data formats
  • Render chemical structures
  • Algorithms for chemical graph theory
  • QSAR descriptors
  • Parts optimized with the YourKit Java Profiler (https://www.yourkit.com/)
