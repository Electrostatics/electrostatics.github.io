NifTools aims to provide users with Tools to View, Edit, Create and Visualise 3D Models stored in the NetImmerse File Format. The NIF format is used by many popular games based on the NetImmerse/Gamebryo Game Engine such as Morrowind, Oblivion, Fallout 3, Fallout: New Vegas and Skyrim. We are creating a unified description of the NIF file format allowing our contributors to build tools and utilities, to facilitate user created content. The tools will allow users to: - View/Edit/Visualise the data of a NIF model. - Libraries to Read/Write the NIF Format. - Import/Export NIF models into various 3D package via Plug-ins. - Custom content creation for NetImmerse/Gamebryo Games.
